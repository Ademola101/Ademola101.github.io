Beginner to web development

Web development is the process of creating webpages and subsequently publishing it on the internet. There are three basic skills needed to design a website, they are namely as follows:
• HTML
• CSS
• JavaScript
HTML which stands for Hypertext markup language deals with the structure and organization of the webpage, while CSS (Cascading Style Sheets)is used to add styles and beauty to the webpages in order to make it readable and user friendly. JavaScript is a programming language used to make the webpage interactive with the user, it is also used to simplify the process of creating website with its arrays of Libraries and Frameworks
(e. g. JQuery, React). It is always advised to learn the 3 skills in the above order (HTML => CSS => JavaScript) Additionally, there are also tools and apps that needs to be installed on your computer (If it is not already installed). They are
1. Text editor
2. Web Browser
A Text editor is simply where you write the codes (HTML, CSS, JavaScript). Most PC come with Text editor such as Notepad. However, there are various advanced text editor known as IDE (Integrated Development Environment) which makes the writing of the code very easy and swift with features such as autocomplete. You are probably familiar with what a web browser is all about. It displays whatever you have done with text editor.
